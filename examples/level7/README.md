# Level 7 Demo

## Install
No separate install required beyond Python 3.10+.

## Setup
From repository root:

python app.py

## Usage
1. Open <http://127.0.0.1:8000>.
2. Click **Level 7**.
3. Watch the HTML output panel for this level's behavior.
