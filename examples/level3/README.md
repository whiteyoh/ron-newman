## How to trigger this example

1. Export your API key and start the app:
   ```bash
   export OPENAI_API_KEY="your_key_here"
   python app.py
   ```
2. Open <http://127.0.0.1:8000>.
3. Click **Level 3**.
4. Review the execution trace shown in the dashboard.
